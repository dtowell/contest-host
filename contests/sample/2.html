Sample Problem 2: Squares

The square of a number is that number multiplied by itself.


Problem

Given a list of integers, compute their squares.

Input

The first line will contain the number of numbers to process.

Each data set consists of a single line containing a single integer less than 10000. At most 100 numbers will be given.

Sample input: 
3
3
4
5

Output

Each data set should produce one line of output containing the square of the original number.

Sample output: 
9
16
25
